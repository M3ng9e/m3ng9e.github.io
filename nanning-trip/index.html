DAY 1 6月19日 周五 ❤ 初见邕城
07:56 — 10:45
广州南 → 南宁东
高铁出发啦~ 车程约3小时，一起听歌、看剧、补觉。
🎵 提前下载好离线内容，准备点小零食路上吃~
11:00 — 11:30
到站 → 前往酒店
地铁40分钟（人均5元）或打车20分钟（人均10元）。先到酒店放行李~
11:30 — 12:30
午餐：第一碗老友粉！ ❤ 必吃
到南宁第一件事！来一碗热腾腾的老友粉：

推荐店铺：
• 复记老友粉 — 老牌名店，酸笋味正宗
• 舒记老友粉 — 汤底浓郁，配料足

点法：汤粉或干捞都行，加猪杂/牛肉，微辣或中辣，再加个卤蛋~
🍜 到了南宁不吃老友粉等于白来！酸笋+豆豉+辣椒的灵魂组合，一碗就爱上
13:00 — 15:30
三街两巷 · 朝阳广场 · 南宁交易场
饭后慢逛消食，三个地方很近：

三街两巷 — 老南宁骑楼街区，拍照超出片
朝阳广场 — 南宁最热闹的商圈
南宁交易场 — 当地人的市场，烟火气十足
📷 三街两巷的骑楼巷子超适合拍合照，随手出片~
15:30 — 16:30
下午茶：老盐水 + 巧八角冰浆
逛累了歇歇脚：
• 老盐水 — 推荐青芒味和西瓜味，酸甜咸冰超解暑
• 巧八角冰浆 — 南宁网红冰品，两个人share一杯
❄ 老盐水推荐青芒味，最经典！冰浆选芒果或百香果味
17:00 — 19:00
亭子码头 看日落 ❤ 浪漫
傍晚的亭子码头最美！邕江边的码头广场：

• 沿江步道散步，吹江风看日落
• 码头广场拍照，华灯初上很出片
• 周边有市集和小吃摊
• 远眺邕江两岸夜景

交通：地铁2号线到亭洪路站，步行约10分钟
🌅 日落时分超浪漫，两个人沿江边走走，看夕阳慢慢落下~
19:00 — 21:30
平西夜市 ❤ 逛吃
从亭子码头步行就到！南宁本地人爱去的夜市：

• 豆腐花 / 烤豆腐 — 嫩滑或外焦里嫩
• 芒果花 — 切花造型，好看好吃
• 各种烧烤串串
• 糖水甜品 / 酸嘢
• 炒螺 / 烤生蚝

两个人每样点一份share，从街头吃到巷尾~
🍰 亭子码头看日落 → 平西夜市逛吃，第一天的完美收尾
21:30 — 23:30
宵夜：雄哥宜州果酱烧烤 ❤ 必吃
晚上7点就开门了！南宁宵夜的灵魂——宜州果酱烧烤：

和普通烧烤不一样，宜州烧烤刷的是酸梅果酱，酸甜开胃，越吃越上瘾。

推荐必点：
• 烤五花肉（果酱味绝了）
• 烤鸡翅
• 烤韭菜
• 烤茄子
• 再来几瓶冰啤酒 🍺

宵夜是南宁的灵魂，必须安排！
🍺 烧烤+啤酒+果酱，南宁宵夜的快乐你想象不到~
23:30
散步回酒店，满足地睡觉~
第一天老友粉+亭子码头+平西夜市+果酱烧烤，吃得太满足了！明天青秀山走起~
DAY 2 6月20日 周六 ❤ 青秀山 + 鸭脚煲 + 琅西夜市
09:00 — 10:00
睡到自然醒 + 早餐
假期不用赶，睡饱了再出发。酒店附近来碗老友粉或卷筒粉~
10:00 — 16:00
青秀山风景区（5A）· 一整天 ❤ 必去
南宁的城市名片，地铁3号线直达（青秀山站B出口）。慢慢逛可以玩一整天~

上午路线：
北门进 → 龙象塔（登顶看邕江全景）→ 观音禅寺 → 千年苏铁园

中午：
景区内有餐厅和小吃摊，简单吃点垫肚子（粽子、玉米、凉粉等）

下午路线：
雨林大观（天然氧吧，超凉快）→ 兰园 → 桃花岛 → 棕榈园 → 西门出

必打卡：
• 龙象塔 — 两人一起登塔看全城
• 观音禅寺 — 东南亚风格佛教建筑
• 雨林大观 — 夏天最凉快的地方
• 兰园 — 热带兰花，适合拍照
• 千年苏铁园 — 上百年的苏铁，震撼
🌿 景区很大！建议坐观光车（20元/人）+ 步行结合。门票20元/人。多带水、防晒、防蚊喷雾！穿运动鞋~
💍 一起爬龙象塔、在雨林散步，累了就长椅上歇歇，享受山里的清凉~
16:00 — 17:30
茶麸洗头 ❤ 体验
逛了一天青秀山，正好去体验南宁特色茶麸洗头：

茶麸（茶籽渣）洗头是广西传统护发方式，天然不刺激，洗完头发又顺又亮。街边小店很多，20-40元/人。

找法：美团/大众点评搜"茶麸洗头"，选评分高的店
💆 两个人一起去体验，洗完头发香香的~ 爬完山洗个头超解乏
18:00 — 19:30
晚餐：老表记鸭脚煲（东盟店）❤ 必吃
你选的这家！南宁人气超高的鸭脚煲。距离青秀山约3-5公里，打车10分钟，很近~

鸭脚软烂入味，汤底鲜辣，两个人点一份煲加配菜就超饱：

• 招牌鸭脚煲（必点！）
• 加腐竹、豆腐泡（吸满汤汁超好吃）
• 一份青菜解腻
• 两碗米饭
🍚 饭点人多，建议提前在美团排号，或者18:00前到
20:00 — 23:00
琅西夜市 宵夜 ❤ 必去
吃完鸭脚煲打车去琅西夜市！你列的这些全都要吃：

• 炒冰 — 现场炒制冰淇淋，芒果味或榴莲味
• 紫苏鸭腿 — 卤鸭腿配紫苏，香到骨头里
• 卷筒粉 — 米皮卷馅淋黄皮酱
• 芒果花 — 切花造型，好看好吃
• 雷公根甘蔗汁 — 清热解暑草本饮料
• 吴家红茶冰 — 红茶冰沙，夏天续命水

两个人分着吃，每样都尝尝~
🍰 芒果花很出片记得拍照！炒冰选芒果或榴莲味最经典
23:00
散步消食，回酒店~
今天走了不少路也吃了不少，回去好好休息，明天最后一天慢慢来~
DAY 3 6月21日 周日 ❤ 海鲜 + 南湖 + 回家
09:00 — 09:30
睡到自然醒 + 早餐
最后一天不赶，慢慢起床。来碗老友粉或卷筒粉~
10:00 — 10:30
出发去淡村海鲜市场
10点出发，打车/地铁前往淡村海鲜市场。到了先逛市场挑海鲜~
10:30 — 12:30
淡村海鲜市场 · 海鲜大餐！❤ 必吃
南宁最大的海鲜市场！最后一顿正餐上海鲜大餐：

怎么吃：
① 市场里逛，挑新鲜海鲜
② 拿到旁边加工店现做（清蒸、椒盐、蒜蓉、爆炒任选）
③ 加工费按斤/份算，十几到几十块

推荐必买：
• 皮皮虾 — 椒盐味，南宁人最爱
• 花蟹 — 清蒸最鲜，蘸醋吃
• 生蚝/扇贝 — 蒜蓉蒸，个大肥美
• 花甲 — 爆炒，配啤酒绝了
• 白灼虾 — 新鲜的虾白灼最甜

人均：海鲜+加工费，两个人150-250元就能吃得很好
🦐 技巧：① 多问几家比价再买 ② 选活蹦乱跳的 ③ 加工店选人多的 ④ 上午去海鲜最新鲜
🦄 两个人一起逛海鲜市场、挑海鲜，超有趣的情侣体验！记得拍海鲜大餐的照片~
13:00 — 15:00
南湖公园 散步 ❤ 悠闲
吃完海鲜打车去南湖公园，免费公园，湖边散步消食~

• 湖边长椅坐坐，聊聊天
• 租个小船在湖上划一划（约30-50元/船）
• 湖心岛逛逛，拍几张合照
• 看看本地人下棋跳舞
🌅 南湖公园很适合情侣散步，节奏慢下来，享受最后的南宁时光~
15:00 — 16:30
找吃的 + 买伴手礼
回市区逛逛，把还没吃到的补上，顺便买伴手礼：

再吃点：
• 再嗦一碗老友粉（真的吃不腻）
• 买杯老盐水路上喝
• 打包一份卷筒粉或酸嘢高铁上吃

伴手礼：
• 老友粉速食装（超市有卖）
• 横县茉莉花茶
• 广西罗汉果
• 酸嘢真空包装
• 壮锦文创小物件
17:00 — 18:00
前往南宁东站
回酒店拿行李，地铁或打车去南宁东站。端午返程人多，别太赶~
⏰ 建议17:00左右出发，预留充足进站时间
19:08 — 22:40
南宁东 → 广州南 · 回家啦~
高铁返程，到广州南站后打车/地铁回家~

📝 实用贴士

两个人出行，这些要注意哦~

🌡 天气穿搭

  • 6月南宁28-34°C，闷热潮湿
  • 穿透气轻薄衣物，短袖短裤
  • 防晒霜SPF50+必涂
  • 端午前后多阵雨，带折叠伞
  • 防蚊喷雾不能忘

🚌 市内交通

  • 地铁1/2/3号线覆盖主要景点
  • 支付宝/微信扫码乘车
  • 两个人打车比地铁贵不了多少
  • 高峰期用网约车提前叫车
  • 青秀山→老表记 打车10分钟

💰 省钱技巧

  • 青秀山门票才20元/人，超值
  • 三街两巷、南湖公园、亭子码头免费
  • 夜市小吃两人share更划算
  • 景点门票提前美团买有优惠

⚠ 避坑提醒

  • 夜市先问价再拿，避免被宰
  • 打车用网约车，避免绕路
  • 端午景区人多，早点出发
  • 生冷食物适量，备好肠胃药
  • 老表记饭点排队长，提前排号

📱 必备APP

  • 12306 / 携程 — 火车票
  • 高德地图 — 导航
  • 美团/大众点评 — 找美食排号
  • 滴滴出行 — 打车

🎉 端午特别

  • 部分景区有龙舟赛、包粽子活动
  • 关注青秀山公众号看活动安排
  • 三街两巷可能有端午民俗展
  • 超市可以买南宁特色粽子带回去
